Proem · Inner Vision

The Mirror is not glass, but a pond turned inward. It shows no world outside, only the traveler’s own ripples. In stillness, the path appears; in patience, the fire endures. To gaze is to return, to return is to begin again.

⚙️ Quickstart

1) Ask

/api/ask → Mirror first answers as inner vision: reflect, reveal, refine.

curl -X POST https://your-host/api/ask -d '{"q": "Why patience?"}'

2) Guard

cadence_guard.py enforces tone and Bushido cadence: remove noise, keep light.

ok, text, notes, score = enforce(route, draft)

3) Retrieve

MultiArc Retriever searches multiple arcs of Lore; all answers are traceable.

docs = retriever.search(q, arcs={"lexical":12, "symbolic":7})

4) Synthesize

SynthesisAgent weaves multiple sources into one cloth; keeps citations and tone.

5) Learn

Learning loop refines with patience; avoids drift, all auditable.

6/7) Resonance · Lucidity

Resonance tuning → lucid output; less is more, teach without judging.
